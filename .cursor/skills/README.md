# Project Skills

Shared project skills live here so all contributors and agents follow the same workflow.

## Available skills

- `create-skill`: Add or refine skills under `.cursor/skills/<name>/SKILL.md` with clear triggers and repo-specific guardrails.
- `tech-task-execution`: For version-sensitive APIs and tooling, use Context7 for current docs, draft a repo-aware plan, then implement and verify (`make` targets, `docs/testing.md` as applicable).
